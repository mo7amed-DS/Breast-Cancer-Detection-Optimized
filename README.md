# Breast Cancer Prediction & Model Optimization

## 📌 Overview
This project aims to classify breast cancer tumors as Malignant or Benign using various Machine Learning algorithms. The core focus was not just building a model, but comparing multiple architectures and fine-tuning the best one for maximum precision.

## 🛠️ Tech Stack
- **Language:** Python
- **Libraries:** Pandas, Scikit-Learn, Matplotlib, Seaborn
- **Environment:** Kaggle / Jupyter Notebook

## 🧪 Methodology
1. **EDA:** Visualizing features and correlations.
2. **Preprocessing:** Scaling data and handling categorical variables.
3. **Model Selection:** Compared SVM, Random Forest, and Logistic Regression.
4. **Optimization:** Used `GridSearchCV` to find the best hyperparameters for the top-performing model.
